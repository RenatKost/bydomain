REST API

Полный контроль над доменами через HTTP запросы. Автоматизируйте проверку, регистрацию и управление DNS.

REST JSON API Key Auth
Получить API ключ
check-domain.sh 
# Проверка доступности домена
curl -X GET \
  "https://api.buydomainbot.com/v1/domains/check?name=example.com" \
  -H "Authorization: Bearer YOUR_API_KEY"

# Ответ
{
  "domain": "example.com",
  "available": true,
  "price": {
    "amount": 9.99,
    "currency": "USD"
  }
}

Аутентификация

Все запросы к API требуют авторизации через Bearer Token. Получите ваш API ключ через Telegram бота командой /api.

Заголовок авторизации 
Authorization: Bearer YOUR_API_KEY

Эндпоинты

Домены

Метод Эндпоинт Описание
GET /v1/domains/check Проверить доступность домена
GET /v1/domains/search Поиск доменов с подсказками
POST /v1/domains/register Зарегистрировать домен
GET /v1/domains Список ваших доменов
GET /v1/domains/{domain} Детали домена
POST /v1/domains/{domain}/renew Продлить домен

DNS

Метод Эндпоинт Описание
GET /v1/domains/{domain}/dns Список DNS записей
POST /v1/domains/{domain}/dns Создать DNS запись
PUT /v1/domains/{domain}/dns/{id} Обновить DNS запись
DELETE /v1/domains/{domain}/dns/{id} Удалить DNS запись

Nameservers и Cloudflare

Метод Эндпоинт Описание
GET /v1/domains/{domain}/nameservers Текущие NS серверы
PUT /v1/domains/{domain}/nameservers Обновить NS серверы
POST /v1/domains/{domain}/cloudflare Подключить Cloudflare
DELETE /v1/domains/{domain}/cloudflare Отключить Cloudflare

Примеры использования

register.py 
import requests

API_KEY = "your_api_key"
BASE = "https://api.buydomainbot.com/v1"

# Регистрация домена
resp = requests.post(
    f"{BASE}/domains/register",
    headers={"Authorization": f"Bearer {API_KEY}"},
    json={
        "domain": "mysite.dev",
        "period": 1
    }
)
print(resp.json())
dns.js 
const API_KEY = 'your_api_key';
const BASE = 'https://api.buydomainbot.com/v1';

// Создание DNS записи
const res = await fetch(
  `${BASE}/domains/mysite.dev/dns`,
  {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      type: 'A',
      name: '@',
      value: '1.2.3.4',
      ttl: 3600
    })
  }
);
console.log(await res.json());

Лимиты и условия

Rate Limits

60 запросов в минуту для обычных ключей. Увеличение лимита по запросу.

Формат ответов

Все ответы в формате JSON. Ошибки возвращают стандартные HTTP коды и описание проблемы.

Безопасность

Только HTTPS. Ключи можно отозвать в любой момент через Telegram бота.
